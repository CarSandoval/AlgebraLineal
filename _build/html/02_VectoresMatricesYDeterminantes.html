2 Vectores Matrices y Determinantes#

Cuando trabajamos con matrices, una de las herramientas fundamentales en álgebra lineal es el determinante. Intuitivamente, el determinante es un número asociado a una matriz cuadrada (una matriz con el mismo número de filas y columnas) que resume información importante sobre dicha matriz.

Por ejemplo:

  • Para matrices 2x2, el determinante se puede visualizar como el área del paralelogramo definido por los vectores fila (o columna).

  • Para matrices 3x3, se relaciona con el volumen del paralelepípedo definido por sus vectores.

  • En general, nos dice si un sistema de ecuaciones tiene solución única, si una transformación lineal es invertible o no, y cómo cambia el “espacio” cuando aplicamos la matriz.

El Nacimiento del Determinante: Un Viaje desde Ecuaciones a Permutaciones#

Históricamente, la invención del determinante no fue un destello de genialidad aislado, sino una evolución gradual que surgió de la necesidad de resolver sistemas de ecuaciones lineales. La elegante definición que hoy conocemos, basada en permutaciones, fue el resultado de un proceso de generalización y búsqueda de un patrón fundamental por parte de matemáticos como Gottfried Wilhelm Leibniz y, posteriormente, formalizado por Augustin-Louis Cauchy.

El Problema Inicial: Resolviendo Ecuaciones#

Todo comenzó con el problema práctico de encontrar una solución general para un sistema de n ecuaciones lineales con n incógnitas.

Demostración: El Origen del Determinante 2x2#

Veamos paso a paso cómo la solución de un sistema simple de 2x2 revela naturalmente la fórmula del determinante.

1. El Sistema de Ecuaciones

Consideremos un sistema general de 2 ecuaciones con 2 incógnitas (x e y):

\[\begin{split}\begin{cases} a_1x + b_1y = c_1 \\ a_2x + b_2y = c_2 \end{cases}\end{split}\]

El objetivo es encontrar una fórmula para x e y usando el método de eliminación.

2. Eliminación de la variable y

Para despejar x, necesitamos eliminar la variable y. Para ello, multiplicamos cada ecuación por un factor que haga que los coeficientes de y sean iguales:

  • Multiplicamos la primera ecuación por \( b_2 \):

    \[ b_2(a_1x + b_1y) = b_2c_1 \implies a_1b_2x + b_1b_2y = c_1b_2 \]

  • Multiplicamos la segunda ecuación por \( b_1 \):

    \[ b_1(a_2x + b_2y) = b_1c_2 \implies a_2b_1x + b_1b_2y = c_2b_1 \]

3. Restando las Ecuaciones

Ahora restamos la segunda ecuación modificada de la primera:

\[ (a_1b_2x + b_1b_2y) - (a_2b_1x + b_1b_2y) = c_1b_2 - c_2b_1 \]

Los términos con y se cancelan:

\[ a_1b_2x - a_2b_1x = c_1b_2 - c_2b_1 \]

4. Despejando x

Factorizamos x del lado izquierdo de la ecuación:

\[ (a_1b_2 - a_2b_1)x = c_1b_2 - c_2b_1 \]

Si el término \((a_1b_2 - a_2b_1)\) no es cero, podemos dividir ambos lados para encontrar x:

\[ x = \frac{c_1b_2 - c_2b_1}{a_1b_2 - a_2b_1} \]

De manera análoga, al eliminar la variable x, encontramos la solución para y:

\[ y = \frac{a_1c_2 - a_2c_1}{a_1b_2 - a_2b_1} \]

5. La Conexión con el Determinante

El denominador, \( a_1b_2 - a_2b_1 \), es crucial. Si este valor es cero, el sistema no tiene una solución única. Este “discriminante” de la existencia de una solución es precisamente lo que se conoce como el determinante de la matriz de coeficientes.

Para la matriz de coeficientes \( A = \begin{pmatrix} a_1 & b_1 \\ a_2 & b_2 \end{pmatrix} \), su determinante es:

\[\begin{split} \det(A) = \left| \begin{matrix} a_1 & b_1 \\ a_2 & b_2 \end{matrix} \right| = a_1b_2 - a_2b_1 \end{split}\]

Esta es la expresión exacta que encontramos en el denominador.

El Salto a 3x3 y la Búsqueda de un Patrón#

Matemáticos como Leibniz, a finales del siglo XVII, se enfrentaron a la tarea de generalizar este proceso. Al resolver un sistema de 3x3, el proceso de eliminación se vuelve mucho más tedioso, pero el denominador común que resulta es:

\[ a_1b_2c_3 + a_2b_3c_1 + a_3b_1c_2 - a_1b_3c_2 - a_2b_1c_3 - a_3b_2c_1 \]

Aquí es donde comienza a surgir la conexión con las permutaciones. Leibniz y sus contemporáneos se dieron cuenta de varias propiedades clave de esta expresión:

  • Composición de los términos: Cada término del polinomio es un producto de n coeficientes (en este caso, 3), donde cada coeficiente proviene de una fila y una columna diferente.

  • El número de términos: Para un sistema 3x3 hay 6 términos (\(3!\)). Para un sistema 2x2 hay 2 términos (\(2!\)). Esto correspondía a \(n!\), el número total de maneras de ordenar (permutar) n elementos.

La Lógica de las Permutaciones y los Signos#

La genialidad estuvo en ver que cada término en la expansión del determinante correspondía a una permutación de los índices de las columnas. Si consideramos los índices (1, 2, 3), cada término es un producto de la forma \( a_{1,\sigma(1)}a_{2,\sigma(2)}a_{3,\sigma(3)} \), donde \( \sigma \) es una permutación del conjunto {1, 2, 3}.

Por ejemplo:

  • Para la permutación identidad \( \sigma = (1, 2, 3) \), obtenemos el término \( a_1b_2c_3 \).

  • Para la permutación \( \sigma = (2, 3, 1) \), obtenemos el término \( a_2b_3c_1 \).

El siguiente desafío era determinar el signo de cada término. Observaron que el signo dependía de la paridad de la permutación (si requiere un número par o impar de intercambios para volver a la secuencia original).

  • Las permutaciones pares (como (1, 2, 3), (2, 3, 1), (3, 1, 2)) tenían un signo positivo.

  • Las permutaciones impares (como (1, 3, 2), (3, 2, 1), (2, 1, 3)) tenían un signo negativo.

El signo de una permutación, denotado como \( sgn(\sigma) \) , es +1 si es par y -1 si es impar.

La Formalización de Cauchy#

Aunque Leibniz sentó las bases, fue Augustin-Louis Cauchy quien, en 1812, formalizó la teoría. Él estableció rigurosamente que el determinante de una matriz A de tamaño nxn es la suma de todos los \(n!\) posibles productos de n elementos (uno de cada fila y columna), donde cada producto se multiplica por el signo de la permutación correspondiente.

En notación moderna, esta es la fórmula de Leibniz:

\[ \det(A) = \sum_{\sigma \in S_n} \text{sgn}(\sigma) \prod_{i=1}^{n} a_{i, \sigma(i)} \]

donde \(S_n\) es el conjunto de todas las permutaciones del conjunto {1, 2, …, n}.
